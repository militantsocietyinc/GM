## Crystal Ball Modifications

The following additions and modifications were made by Bradley Bond
for the Crystal Ball project (https://github.com/bradleybond512/crystal-ball):

- macOS native Tauri desktop application shell
- Air Strikes & Drone Monitoring panel (ACLED integration)
- Peace / Finance / War monitoring modes with auto-triggers
- Cyber Threat Intelligence panel (IOC feeds)
- Alert Center with persistent signal history
- World Bank country profile enrichment
- AI Summary overlay for all panels
- In-app Help guide
- Check for Updates from macOS Help menu
- macOS Window menu (Minimize, Zoom, Close)
- Open Logs Folder menu item
- Low Power Mode toggle
- Removal of all license key / premium feature gating
- Satellite and terrain basemap options
- Intelligence & OSINT live news channel category
- YouTube account sign-in support
